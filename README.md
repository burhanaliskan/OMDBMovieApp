# OMDBMovies App
<br />


## Features

- [x] Storyboard
- [x] UIKit
- [x] Datasource
- [x] Protocol and Delegate
- [x] URLSession with OMDBApi & Codable
- [x] Cocoapods
- [x] Firebase

## Requirements

- iOS 14.5+


## Design Patterns

- MVC Design Pattern


## Library

- Kingfisher 6.0
- Alamofire  5.2
